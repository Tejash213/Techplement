# Quiz Generator

This is a simple command-line quiz generator in Java.

## Features
- Add multiple-choice questions.
- Take the quiz and receive feedback.
- Score calculation.

## How to Run
1. Compile:
   ```
   javac QuizGenerator.java
   ```

2. Run:
   ```
   java QuizGenerator
   ```
